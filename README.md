# Simple Mern Auth
## Aplikasi apa ini?
Latihan stack MERN dengan studi kasus autentikasi Login dan Register. Materi dari tutorial: https://youtube.com/playlist?list=PL_mAvlfqBUVKEfjKp5GjNIg0q-bkXNkgs

## Teknologi yang digunakan
- ReactJS
- ExpressJS
- MongoDB
- Bootstrap
- bcryptjs
- JWT

## Cara Penggunaan
### Setup Frontend
- `cd frontend`
- `npm run start`

### Setup Backend
- `cd backend`
- `npm run start-dev`
